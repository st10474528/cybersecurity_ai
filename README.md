# 🔒 Cybersecurity Awareness Chatbot - NECH

## Project Overview

This is a command-line cybersecurity awareness chatbot developed for South African citizens as part of a national cybersecurity campaign. The chatbot educates users about online safety, phishing scams, password security, safe browsing habits, and other critical cybersecurity topics.


## Features

- 🎵 Voice greeting on application startup
- 🎨 ASCII art logo display
- 👤 Personalized user interaction with name collection
- 💬 Intelligent responses to cybersecurity questions
- ✅ Input validation for empty or invalid entries
- 🎨 Colored console output for better readability
- ⌨️ Typewriter text effect for natural conversation feel
- 🔐 Comprehensive cybersecurity education content
- 🇿🇦 South Africa specific security advice

## Requirements

### Prerequisites
- Windows Operating System (required for audio playback)
- .NET 8.0 SDK or later
- Git (for cloning the repository)

### Dependencies
- System.Media (built into .NET Framework)
- No external NuGet packages required
